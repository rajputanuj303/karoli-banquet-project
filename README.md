# Karoli Banquet Project

A full-featured banquet hall management and booking platform built with the MERN stack (MongoDB, Express.js, React, Node.js).

## Structure

- `backend/` - Express server, MongoDB models, API routes
- `frontend/` - React app (user-facing + admin panel)

## Setup

1. Install dependencies in both `backend` and `frontend` folders
2. Start backend server
3. Start frontend React app

## Features

- Hall listing, packages, gallery, booking flow, admin panel, integrations
